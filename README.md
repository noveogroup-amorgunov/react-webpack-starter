## react-webpack-starter

Example of using reactJS with webpack (es6, jsx).

### Install

```
git clone https://github.com/noveogroup-amorgunov/react-webpack-starter.git
cd {path}/client
npm install
cd {path}/server
npm install
```

### Build

```
cd {path}/client
webpack --watch
```

### Run server:

```
cd {path}/server
node index
```
